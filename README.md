# 🎮 Simon Says Game

An interactive memory-based browser game built using HTML, CSS, and JavaScript.

## 🧠 About This Project

This project was created to practice:
- DOM manipulation
- Event handling
- Game logic implementation
- Working with arrays and sequences
- Improving UI/UX design

## 🚀 Features

- Random color sequence generation
- Level progression
- High score tracking
- Smooth animations
- Responsive design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## 🎯 How to Play

1. Click the Start button or press any key.
2. Watch the flashing color sequence carefully.
3. Repeat the sequence in the correct order.
4. The game ends if you press the wrong color.
5. Try to beat your highest score!
